A mobile application for creating and managing professional resumes on the go.
This Android app offers a user-friendly interface for users to input personal information,
work experiences, education details, skills, languages, and hobbies.
It generates customizable, visually appealing resumes from the data and allows users to select templates for their CVs.
The app also supports exporting resumes to popular formats like PDF and DOCX.
